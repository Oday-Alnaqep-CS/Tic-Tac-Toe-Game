### 📝 Project Overview
This project demonstrates **state management** and **matrix logic** in C#. It features a robust algorithm that scans the 3x3 grid for horizontal, vertical, and diagonal win patterns in real-time, showcasing clean code practices.

### 🚀 Key Logic Features
* **Smart Win Algorithm:** Efficiently detects all 8 possible winning combinations.
* **Dynamic Turn System:** Manages seamless transitions between Player X and Player O.
* **Neon UI Aesthetics:** High-contrast dark mode with glowing status indicators, reflecting a professional visual style.
* **State Reset:** Intelligent board clearing and score tracking for continuous play.

### 🛠️ Technical Implementation
- **Grid Indexing:** Utilizing a coordinate-based system for the 3x3 layout.
- **Event Unification:** Using a central event handler to ensure clean, DRY (Don't Repeat Yourself) code.
- **Visual Feedback:** Dynamic color shifts to highlight winning paths and enhance user experience.
